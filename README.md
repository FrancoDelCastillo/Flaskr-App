# Flaskr-App

Despues de clonar 

instalar virtual environment en la carpeta raíz(Flaskr-App):
python3 -m venv venv

Activamos (venv):
source venv/bin/activate

Luego instalamos flask:
pip install flask

Inicializamos el database:
flask init-db

Por último corremos la app:
flask run

Abrimos el navegador en http://127.0.0.1:5000/
